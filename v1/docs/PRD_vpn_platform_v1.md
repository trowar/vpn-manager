# 分布式 VPN 平台 V1 产品需求文档（PRD）

## 一、产品目标
建设一个分布式 VPN 平台，采用 Web 控制端与 VPN 节点端分离部署方式，支持 OpenVPN、WireGuard、邮箱注册登录、订单提交、管理员确认开通、节点自动部署、单账号单会话、节点异常切换。

## 二、角色
### 普通用户
- 注册
- 邮箱验证
- 登录
- 查看套餐
- 提交订单
- 取消未完成订单
- 查看订单
- 查看当前权益
- 下载配置
- 查看已用量、剩余量、到期时间
- 修改自己的密码
- 找回密码

### 管理员
- 统一登录
- 首次强制修改默认密码
- 查看待处理订单
- 查看用户
- 删除用户
- 修改用户密码
- 查看用户权益
- 查看用户订单
- 取消未完成订单
- 添加服务器
- 自动部署节点
- 查看部署报告
- 查看节点状态
- 配置 Telegram 联系方式
- 配置开放注册开关
- 配置新用户赠送策略
- 配置订单过期时间

## 三、系统初始化
- 默认管理员：admin / admin
- 首次登录必须修改密码
- 未修改前不得访问其他后台页面

## 四、注册与登录
- 注册需要：邮箱、密码、邮箱验证码、图片验证码
- 登录页始终显示图片验证码
- 找回密码页显示图片验证码
- 必须完成邮箱验证后才能登录
- 邮箱验证码有效期：10 分钟
- 同一邮箱 60 秒内不得重复发送
- 同一邮箱每天最多发送 10 次
- 未完成邮箱验证的账号 24 小时后自动删除
- 用户修改密码成功后，Web 会话全部失效，VPN 会话全部断开，需重新登录

## 五、开放注册
- 支持后台关闭开放注册
- 关闭后隐藏注册页

## 六、套餐与权益
V1 仅支持：
- 免费权益
- 时长包
- 流量包

V1 不支持混合套餐。

规则：
- 时长包默认不限流量
- 流量包默认不限使用期限
- 新用户赠送由管理员后台配置
- 可同时赠送流量和时长
- 赠送在邮箱验证成功后发放
- 赠送规则变更仅影响后续新用户
- 不支持管理员手动补送权益

## 七、订单系统
- 用户必须先选套餐再下单
- 用户可取消未完成订单
- 管理员可取消任何未完成订单
- 管理员点击确认后，系统按订单套餐内容自动开通
- 确认时不允许改套餐内容
- 订单过期时间支持后台全局设置
- 默认 24 小时
- 仅影响新订单
- 用户刷新页面后看到开通结果

## 八、用户前台展示
首页展示：
- 当前套餐
- 已用量
- 剩余量
- 到期时间
- 各权益信息

流量显示规则：
- 小于 1GB 显示 MB
- 大于等于 1GB 显示 GB

流量耗尽后：
- 立即断开 VPN
- 仍允许登录网页
- 仍允许查看历史
- 仍允许下单

配置页：
- OpenVPN 与 WireGuard 左右并列显示
- 即使无权益，也显示两个协议栏目，但下载按钮置灰

## 九、VPN 会话规则
- 单账号全局只允许一个在线会话
- 跨协议统一限制
- 若已有会话在线，新连接拒绝
- 拒绝文案：已有连接未断开，请稍后再试

## 十、配置文件规则
- 支持 OpenVPN 和 WireGuard
- 每次下载均实时生成最新配置文件
- 配置文件应尽量使用稳定入口，不强绑定单个节点
- 不支持管理员单独关闭某个协议

## 十一、服务器与节点
- 部署顺序：
  1. 先部署 Web 控制端
  2. 管理员登录后台
  3. 添加服务器
  4. 检测通过后立即自动部署
  5. 生成部署报告

- 新增服务器时需填写：
  - IP
  - SSH 端口
  - SSH 用户名
  - 密码或私钥

- 部署失败后保留在服务器列表中
- 仅允许管理员手动重试
- 默认心跳超时阈值：60 秒
- 异常节点包括：
  - 节点离线
  - 节点心跳超时
  - 节点部署失败

- 节点分配主策略：在线用户最少
- 并列时：最近最少分配
- 前台异常提示：当前节点异常，系统正在切换

## 十二、用户删除
- 用户删除采用真删除
- 删除内容包括：用户、订单、权益、配置、日志
- 删除时要求输入用户邮箱二次确认
- 删除后原邮箱可重新注册

## 十三、前端 UI 要求
- 统一采用左侧导航栏 + 右侧内容显示区
- 用户端建议菜单：
  - 首页
  - 套餐
  - 订单
  - 配置
  - 个人中心
- 管理端建议菜单：
  - 仪表盘
  - 订单
  - 用户
  - 服务器
  - 节点
  - 设置

管理员首页内容排序：
1. 待处理订单
2. 异常节点
3. 即将到期权益
4. 用户总览

## 十四、技术约束
- 数据库：SQLite

README 开头必须提供 Linux 一键安装方式，支持：

```bash
wget -O - <INSTALL_URL> | bash
```

或

```bash
curl -fsSL <INSTALL_URL> | bash
```

安装脚本需支持：
- apt
- yum

并自动完成：
- 依赖安装
- 控制端部署
- 数据库初始化
- 默认管理员初始化

安装完成后需打印：
- 登录地址
- IP
- 端口
- 默认账号密码
- 初始化提示

## 十五、验收标准
1. 可通过一键安装完成控制端部署
2. 支持 apt 和 yum 环境
3. 安装完成后打印登录入口
4. 默认管理员可登录并首次强制改密
5. 用户注册必须经过邮箱验证
6. 注册、登录、找回密码均含验证码机制
7. 用户可查看套餐并提交订单
8. 管理员可确认订单并自动开通
9. 用户可下载 OpenVPN 和 WireGuard 配置
10. 系统只允许单账号一个在线会话
11. 节点支持自动部署与状态管理
12. 节点异常时前台有明确提示
13. 删除用户时可彻底删除全部相关数据
