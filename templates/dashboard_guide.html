使用说明：如何设置 VPN

  1. 选择客户端类型：WireGuard 或 OpenVPN。
  2. 进入“控制台首页”确认订阅状态为“已生效”。
  3. 下载对应配置文件：WireGuard 与 OpenVPN 统一为“全局代理”模式（`.conf` / `.ovpn`）。
  4. 在客户端中导入配置文件并连接。
  5. 连接成功后即可使用。

WireGuard

速度快、配置简单，统一使用全局代理；支持二维码导入。

一键下载 WireGuard 客户端 全部 WireGuard 下载入口

OpenVPN

兼容性广，统一使用全局代理 `.ovpn` 配置。

  1. 打开 OpenVPN Connect 客户端。
  2. 如果显示 “Connect with Server URL or Cloud ID” 页面，点击下方 “Have a configuration file instead (.ovpn)?”。
  3. 点击 “Upload File”。
  4. 在文件选择框中定位到下载目录，选择 `ovpn-用户名-global.ovpn`。
  5. 点击“打开”，导入成功后点连接即可使用。
一键下载 OpenVPN 客户端 全部 OpenVPN 下载入口

常见问题：连接不上时，优先检查套餐有效期、设备时间是否准确、是否被本地防火墙拦截。
