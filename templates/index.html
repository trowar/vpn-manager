VPN 门户（WireGuard + OpenVPN，支持 USDT 付费）

注册后可直接登录。用户下单并使用 USDT 转账，提交交易哈希后系统会通过 Webhook 自动核验到账并开通或续费 VPN。 客户端支持 WireGuard 与 OpenVPN 两种协议。

注册账号 登录控制台 WireGuard 客户端 OpenVPN 客户端

开通流程

1. 注册并登录

用户使用邮箱注册后可立即登录，无需人工审核账号。

2. 创建 USDT 订单

可选择管理员配置的套餐（按时长或按流量），系统自动生成应付金额和收款地址。

3. 支付并提交 TxHash

完成链上转账后提交交易哈希，等待 Webhook 自动核验或管理员兜底处理。

4. 下载配置接入 VPN

订单确认后自动开通，可下载 WireGuard `.conf` 或 OpenVPN `.ovpn` 配置。

如何设置 VPN（新手说明）

  1. 在手机或电脑安装 WireGuard 或 OpenVPN 客户端。
  2. 登录用户控制台，确认订阅状态为“已生效”。
  3. 点击下载对应配置（WireGuard `.conf` / OpenVPN `.ovpn`）。
  4. 在客户端中导入配置文件并连接。
  5. 首次连接成功后即可访问外网。

如果连接失败，请先检查：套餐是否生效、设备系统时间是否准确、网络是否允许 UDP。
