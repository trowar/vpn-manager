OpenVPN 使用说明（导入 .ovpn）

  1. 先在“用户首页”下载 OpenVPN 全局代理配置文件。
  2. 打开 OpenVPN Connect，若看到 “Connect with Server URL or Cloud ID” 页面，点击 “Have a configuration file instead (.ovpn)?”。
  3. 点击 “Upload File”。
  4. 在文件选择窗口中进入下载目录，选中 `ovpn-用户名-global.ovpn`。
  5. 点击“打开”完成导入，再点击连接按钮即可。

如果导入后无法连接，先确认套餐已生效、系统时间正确、防火墙未拦截 VPN 连接。
